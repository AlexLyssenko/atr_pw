Playwright Crash Course

Learn how to use Playwright for robust end-to-end testing with a modern, practical approach.

Course Outline

  1. Environment Setup (Approx. 0.5 hour)

    Get your workspace ready by learning how to handle different package managers.

    • npm
    • yarn
    • pnpm

  2. Monorepo with Nx (Approx. 1 hour)

    Understand the benefits of a monorepo and set up your project using the Nx toolkit.

    • What is a monorepo?
    • Getting started with Nx
    • Creating a project structure

  3. Testing Frameworks (Approx. 1.5 hours)

    Compare testing frameworks and integrate Playwright into your project.

    • Overview of test types
    • JEST vs. Playwright: Key differences
    • Adding Playwright to your project

  4. Automating External Sites (Approx. 1 hours)

    Learn to test external applications by setting up a server and generating tests.

    • Add an mcp server
    • Generate and run tests with a prompt

  5. Testing Your Own Application (Approx. 2 hours)

    Use locators to interact with elements on your own site and create targeted tests.

    • Understand and add locators
    • Generate and run tests based on prompts
